class MeadowRawQuery

Meadow Raw Query Library

This library loads and stores raw queries for FoxHound to use. You can overload the default query that is built for each of the following query archetypes:

Create, Read, Reads, Update, Delete, Count

You can also load other custom queries and give them an arbitrary name.
