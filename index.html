Every truth has two sides; it is as well to look at both, before we commit ourselves to either.

Aesop

Data contains so many stories. With the right tools, developers can enable users to be amazing storytellers. We can even tell stories of our own!

Fable: Core Services

It is tiring to setup logging and settings management in every application you write. Fable provides a single line solution to have simple logging to the console via bunyan. Add a simple configuration object and it can also write the log to a file. Or even mongodb!

Orator: Restful Web Server

We think restify is pretty awesome. But using it in each application server or microservice turned into the same 25 or 50 lines of boilerplate code. With Orator you can just configure what you want.

Meadow: Data Access & Automatic Endpoints

Who doesn't love writing the same code over and over again?

Foxhound: Programmatic Query Generation

Write the same code. Access mongo, mysql or whatever.

Pict: Markdown Inspired Data Definition Layer

Scribble a few words down and voila! Database create statements, documentation, graphs, mock objects for unit tests and a bunch more. Plays well with meadow.
